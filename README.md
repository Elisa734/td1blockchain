# td1blockchain
Ce repository regroupe tous mes exercices pratiques sur les concepts fondamentaux de la Blockchain réalisés en Python.

🎯 Objectif du projet
Comprendre les bases techniques de la blockchain en implémentant :

Le fonctionnement de la fonction de hachage SHA-256
La création d’un arbre de Merkle
La sensibilité des hashes (effet avalanche)
Les notions de double hachage et d’immuabilité


📋 Exercices inclus
ExerciceDescription1.1Calcul de hash SHA-256 de plusieurs mots
1.2Comparaison entre "Bitcoin" et "bitcoin" (effet avalanche)
1.3SHA-256 simple vs Double SHA-256
1.4Recherche d’entrée correspondant à un hash cible
2.1aCalcul des hashes individuels des transactions
2.1bcConstruction complète d’un Arbre de Merkle
2.4H33 : Double hachage d’une même transaction

🛠 Technologies utilisées
Python 3
hashlib (bibliothèque native pour SHA-256)
Programmation orientée exercices et visualisation claire

🚀 Comment exécuter les exercices ?

Clone le repository :

Bashgit clone https://github.com/ton-username/blockchain-exercices.git ou telecharge le dossier

Lance le programme principal (menu interactif) :
 python main.py dans le terminal.
 
Ou exécute chaque exercice individuellement :
Bashpython exercice1_1.py
python exercice2_1_bc.py

🎓 Ce que j’ai appris

Comment fonctionne le hachage cryptographique (SHA-256)
L’importance de l’immuabilité des données
Le rôle central de l’Arbre de Merkle dans la blockchain
Pourquoi un tout petit changement d’entrée produit un hash complètement différent
Les bases techniques nécessaires pour comprendre Bitcoin et les blockchains
